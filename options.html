ℹ️ About

Privacy Shield v1.0.0

A privacy-focused browser extension that blocks tracking scripts and protects your online privacy.

This extension uses declarativeNetRequest API for efficient, high-performance tracker blocking.


Features:

  • Real-time tracker blocking
  • Categorized blocking (ads, analytics, social, etc.)
  • Customizable whitelist and blacklist
  • Detailed statistics and analytics
  • Low memory and CPU footprint
